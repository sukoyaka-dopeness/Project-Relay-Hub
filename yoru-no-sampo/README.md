# 夜の散歩 — Night Walk (MVP)

深夜に眠れない人が、  
ただ静かに夜道を歩き、  
“だれかも起きている気配”を感じられるサービスです。

- チャットなし  
- 出会い系要素なし  
- 匿名  
- ただ歩くだけ  
- ただ気配を感じるだけ  

夜の静けさと、干渉しない優しさを大切にしています。

---

# 🌙 コンセプト

## 静けさ  
ユーザーに何かを「させる」ことを避け、  
UIは控えめ、音はなし。  
夜の空気を壊さないことが最優先。

## 気配  
他ユーザーは「点」や「淡い動き」で存在を示すだけ。  
名前もメッセージも表示しない。  
“誰かがいる”という淡い感覚だけを残す。

## 儀式化しない  
「ようこそ」「始めましょう」などの誘導文は排除。  
ロード画面も退出画面も、静かで控えめ。

---

# 🌙 体験の流れ（UI Flow）

1. **アクセス**  
   → ロード画面へ

2. **ロード画面（ランダム文言）**  
   - 「誰かも起きているようです」  
   - 「遠くで気配がします」  
   - 「静かな夜です」  
   - 「深夜の空気ですね」  
   - 「こんばんは」  
   - 「夜ですね」

3. **時間帯チェック**  
   - 深夜（23:00〜05:00）→ 通常入室  
   - 昼 → 「深夜専用です」＋テストモード

4. **気分アイコン選択**

5. **散歩画面（メイン）**  
   - 右タップ → 右へ歩く  
   - 左タップ → 左へ歩く  
   - スタンプ（20種）  
   - 定型文（30種）  
   - コメント連打で羊  
   - 他ユーザーの気配  
   - AIキャラの短い一言  
   - 広告（自販機／コンビニ／看板）

6. **退出（任意）**  
   - 「また夜が来たら」  
   - 「おやすみなさい」

7. **終了演出**  
   - **流れ星が1本だけ静かに横切る → 暗転**

# 🌙 夜の風景（Canvas）

Canvas 1枚で以下のレイヤーを順に描画：

```
Layer 0: 夜空（グラデーション）
Layer 1: 遠景（建物・山）
Layer 2: 星（田舎道のみ）
Layer 3: 街灯（光源＋ポール）
Layer 4: 道（アスファルト）
Layer 5: 広告（自販機・看板）
Layer 6: 他ユーザーの気配（点）
Layer 7: スタンプ・定型文
Layer 8: 流れ星（終了演出）
```

### 道の種類
- **住宅地**：街灯が等間隔  
- **田舎道**：星が見える  
- **商店街**：看板が多い  

### 夜空
- 基本は濃紺  
- **04:30〜05:00 の間だけゆっくり明るくなる**

### 星
- 田舎道のみ  
- 都市部では表示しない

### 街灯
- 稀に点滅  
- 朝に消灯
