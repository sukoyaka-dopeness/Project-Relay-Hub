# Night Lantern Stroll (仮タイトル)

深夜限定の「眠れない人たちの静かな気配共有」ウェブサービス（PWA）。

## コンセプト
- 目的: 話すより「同じ時間に起きている他者の気配」を感じて安心する
- 深夜限定（ユーザー申告の23:00〜5:00のみ入室可）
- 出会い系厳禁: 文字チャットなし、スタンプ＋定型5文字のみ、DM/プロフィールなし
- 世界観: 夜の街を散歩するイメージ（時間軸バー＋アイコン＋看板広告）

## 技術スタック（MVP）
- Frontend: Vanilla JS + PWA (Service Worker)
- Backend: Firebase Realtime Database + Anonymous Auth
- リアルタイム: Presence管理で自動退出

## 現在の仕様まとめ
- インタラクション: スタンプ（20種） + 定型5文字選択（30種）
- 環境音: MVPは「夜の街」1テーマ、時間帯で自動変化（23時やや賑やか → 4時ほぼ静寂）
- 広告: 夜の自動販売機/コンビニ看板として風景の一部
- 隠し機能: 「…」スタンプ5回連打で羊カウント（自分だけ見える、時間経過でフェードアウト）
