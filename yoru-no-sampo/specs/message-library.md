# MESSAGE LIBRARY

## Purpose
This document contains the full library of messages available in the night walk service. It separates the message content from the system specification so the list can expand without changing core mechanics.
## 目的
このドキュメントは夜の散歩サービスで使用されるメッセージの完全ライブラリを収録する。メッセージ内容をシステム仕様から分離し、将来メッセージが増えてもコア仕様を変更せずに拡張できるようにする。

# Message Design Principles
## English
Messages should be short, calm, and emotionally neutral or gentle. They represent quiet presence rather than conversation. Messages should avoid strong opinions, arguments, or anything that breaks the atmosphere of a quiet night.
## 日本語
メッセージは短く、穏やかで、感情的に強すぎないものにする。目的は会話ではなく「静かな存在の共有」である。強い主張や議論を生む内容、夜の雰囲気を壊すものは避ける。
