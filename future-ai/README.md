# おやじストッパー

![おやじストッパーロゴ](logo.png)  
*おやじギャグ連発中 → AIが「はい1回目～！」と笑顔でツッコミ*

## 🎯 企画概要
おやじギャグ、昭和トーク無限ループ、絵文字洪水を**優しくツッコむ相棒AI**。  
深い感情介入せず、**ユーモアだけで周囲の疲弊を防ぎます**。

日本の高文脈文化に最適化。**恥をかかせず、笑いで解決**。

Before: 「またその話…(心の声)」
After: AI「はい1回目～！」→一同笑い


## 🤔 なぜこれが必要か？

| 問題 | 従来の解決 | おやじストッパー |
|------|------------|----------------| 
| おやじギャグ連発 | 無視/我慢 | 「はい1回目～！」と軽くツッコミ |
| 昭和トークループ | 話題変更困難 | 「その話今週5回目です」と実況 |
| 絵文字洪水 | チャット崩壊 | 「絵文字祭りですね！」と称賛 |
| 本人無自覚 | 指摘しづらい | AIが相棒として自然に伝達 |

**結果**: 周囲のストレス30%減、本人は笑って自覚。

## 🚀 主な機能

- **リアルタイムおやじ検知**  
  `おやじギャグ3連続 → 「はい1回目～！」`

- **昭和度メーター**  
  `キーワード密度75% → 「昭和度危険水域です！」`

- **自動ギャグストック**  
  `ネタ切れ時 → 「次はこのダジャレどう？」`

- **P2Pギャグ交換**  
  `友達と新ネタ交換（AirDrop/Quick Share/WebRTC）`

- **アバター6種**  
  `冷徹秘書 / 若者後輩 / 昭和司会者 / 孫キャラ / おかん / ギャル後輩`

- **落ち込み寄り添い**  
  `声トーン低下 → 「昭和話なら私が聞きますよ」`

## 🎮 クイックスタート
```bash
# 1. リポジトリクローン
git clone https://github.com/sukoyaka-dopeness/oyaji-stopper.git
cd oyaji-stopper

# 2. 依存関係インストール
npm install
# または Expo Goで即開始

# 3. 開発サーバー起動
npx expo start
# QRコードをExpo Goでスキャン

🛠️ 技術スタック

Frontend: React Native + Expo + Tamagui
AI: Llama3.2 1.5B (on-device)
音声: Whisper Tiny + Web Speech API
P2P: WebRTC + Simple-Peer
フォント: 創英角ポップ体風

🎨 デザイン方針
フォント: 創英角ポップ体（昭和ノスタルジー）

カラー: ピンク基調 + ラメグラデーション（女性誌表紙風）

トーン: 飲み屋の相棒（親しみ > 指導）


📈 コアメトリクス
指標	計算式	安全圏
昭和度	昭和キーワード/総単語	<70%
おやじ指数	ギャグ×3+繰り返し×2+絵文字	<100
ギャグ連発	ダジャレ/10分	<5回
🤝 貢献方法
